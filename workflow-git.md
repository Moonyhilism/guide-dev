- branch : Développer des fonctionnalités isolées les unes des autres.

- commit : Ceci est une étape dans l'historique d'un projet, que l'on va pouvoir identifier avec un message.

- push : Transfert d'un dépôt local vers un dépôt distant.

- pull request : Transfert d'un dépôt distant vers un dépôt local.

- merge : Fusionner deux commits en un seul dépôt.
